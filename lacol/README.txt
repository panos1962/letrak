Το παρόν directory αποτλεί ένα είδος οδηγού για διάφορα files που δεν θέλουμε
να κοινοποιηθούν· τέτοια files είναι συνήθως διάφορα configuration και data
files που θα βρούμε συνήθως στα directories "local", "private" κλπ. Εννοείται
ότι τα πρότυπα files που θα δημιουργήσουμε στο παρόν directory δεν πρέπει σε
καμία περίπτωση να είναι ακριβή αντίγραφα των αντίστοιχων production files
και δεν πρέπει σε καμία περίπτωση να περιέχουν ευαίσθητα ή απόρρητα δεδομένα 
π.χ. user names, passwords κλπ.
